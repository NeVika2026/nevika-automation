# FAMALL

Модульная HTML/CSS/JavaScript-структура интернет-магазина FAMALL.

Текущий этап: каталог наполняется из `data/products.json`, товары импортируются из PDF-прайса и источников FAMALL, добавлены коллекции `Новинки`, `Хиты продаж`, `Акции` и отчет по товарам без реального фото.

## Структура

- `index.html` - главная страница.
- `pages/` - внутренние страницы сайта.
- `assets/css/` - базовые стили, компоненты и стили страниц.
- `assets/js/` - ядро, модули и интеграции.
- `assets/images/` - изображения.
- `assets/icons/` - иконки.
- `data/` - статические данные для каталога и новостей.
- `docs/sitemap.md` - карта сайта.

## MVP данных и админки

- `data/products.json` - тестовые товары для каталога, карточки товара и корзины.
- `data/categories.json` - тестовые категории.
- `data/missing-images-report.json` - товары без реального фото, для которых созданы премиальные заглушки.
- `missing-images-report.xlsx` - Excel-отчет по товарам без изображений.
- `data/prices.json` - тестовый прайс России.
- `data/news.json` - тестовые новости.
- `admin/index.html` - заглушка админ-раздела с принципом минимального участия: загрузить каталог, загрузить прайс, нажать `Обновить каталог`.
- `robots.txt`, `sitemap.xml`, `404.html`, `site.webmanifest` - файлы для публикации статического сайта.
- `DEPLOY.md` - краткая инструкция по публикации.

Будущий автоматический сценарий обновления каталога:

1. Сопоставить товары по артикулу и названию.
2. Создать новые карточки товаров.
3. Обновить розничные и партнерские цены.
4. Скрыть отсутствующие товары.
5. Обновить категории и публичный каталог.
