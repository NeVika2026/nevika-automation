FAMALL Admin MVP

Обновление контента

Минимальный сценарий: загрузить каталог FAMALL, загрузить прайс России и нажать одну кнопку обновления.

Обновить каталог сайта

После нажатия система должна сама создать новые карточки, обновить цены, скрыть отсутствующие товары, обновить категории и публичный каталог.

Что система сделает автоматически

  1. Сопоставит товары из каталога и прайса по артикулу, затем по названию.
  2. Создаст карточки для новых товаров.
  3. Обновит розничные и партнерские цены.
  4. Скроет товары, которых больше нет в прайсе или наличии.
  5. Обновит категории и витрину каталога.

Ошибки и предупреждения

  • Обработка файлов пока не подключена: сейчас это UX-MVP.
  • После backend-этапа отчет покажет только товары, которые требуют внимания.

Новости
